Get Started

This page is has a collection of examples that you can take using both Syncano and the Angular 1.x Library. It connects the two using the ng-syncano library found here. This library makes it possible to use the Syncano API without having to require('') it or include the library in your html header section.

To download one of the apps, visit this github page and grab the dist/ folder.
Note: You will need a Syncano account to use these apps!
